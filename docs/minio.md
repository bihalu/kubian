# MinIO
MinIO Server console is launched on node port 30001  
and s3 endpoint is on node port 30002.

## Build
```bash
cd ~/kubian/apps
./kubian-minio-2023.7.18.sh
```

## Install
```bash
./kubian-minio-2023.7.18.tgz.self install
```

## Uninstall
```bash
./kubian-minio-2023.7.18.tgz.self uninstall
```
